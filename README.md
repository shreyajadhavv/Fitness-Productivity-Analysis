# Fitness-Productivity-Analysis
Data analysis project on fitness and productivity  using survey data 
Fitness and Productivity Tracking Survey 
Project Overview

This project analyzes lifestyle habits and their impact on productivity, concentration, and mood. Data was collected through a Google Form survey.

 Tools Used

Google Forms (Data Collection)

Excel (Data Storage)

Python (Pandas, Matplotlib, Seaborn)

Google Colab (Data Analysis)

 Dataset Features

Gender

Age Group

Sleep Duration

Water Intake

Coffee Consumption
Steps Walked

Exercise Duration

Workspace

Concentration Rating

Mood Rating

 Key Insights

Sleep and water intake positively impact concentration.

Music improves mood but may slightly reduce focus.

Coffee consumption is associated with exercise habits.
